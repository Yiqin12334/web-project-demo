# web-project-demo
前端HTML/CSS/JS交互功能实战项目

## 项目介绍
本项目是一个[交互式图库网站[，实现了[html ,css]等核心功能，注重前端交互体验和逻辑处理，适合前端基础实战练习。

## 运行步骤
1. 克隆仓库到本地：`git clone https://github.com/Yiqin12334/web-project-demo.git`
2. 进入项目根目录，用浏览器打开`index.html`文件即可运行；若使用VS Code，可安装Live Server插件启动本地服务，体验更流畅的热更新效果。

## 核心技术点
1. **HTML5**：使用`<form>`表单标签及`input`各类属性（如`required`/`pattern`）构建交互界面；
2. **CSS3**：采用Grid布局划分功能区域，使用`::before`/`::after`伪元素美化表单和按钮样式，通过`animation`实现操作反馈动画；
3. **JavaScript**：
   - 实现表单数据的实时验证（如手机号、邮箱格式校验）；
   - 通过事件委托处理动态生成元素的交互逻辑，优化性能。
